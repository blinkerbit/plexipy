How to Use

  1. Enter your Azure AD App Registration details above
  2. Click "Get Access Token" to acquire a token
  3. The token will be stored in your browser and can be reused
  4. Click "Use in POV App" to go to the POV app with the token pre-filled
  5. Save configurations for quick access later (secrets are stored locally)

Note: Access tokens typically expire after 1 hour. The token is stored in your browser's localStorage and persists across sessions. Your credentials are only sent to the backend for token acquisition.
